# 输入框调整为高而窄的比例

**日期**: 2026-06-27
**类型**: feature
**影响范围**: frontend

## 设计意图

输入区应位于屏幕下 1/3 处，高而窄——像一个专注的命令中心。视觉重心自然落在底部输入区，符合人机对话的视线流动。

## 修改内容

| 属性 | 修改前 | 修改后 | 变化 |
|------|--------|--------|------|
| `--input-bar-height` | 64px | **100px** | +56% |
| `--max-content-width` | 960px | **720px** | -25% |
| `.input-bar` padding | 8px 20px | **20px 24px** | 上下大幅增加 |
| `.chat-input` min-height | 42px | **52px** | +10px |
| `.chat-input` font-size | 14px | **15px** (`--text-md`) | +1px |
| `.send-btn` | 38×38px | **46×46px** | +8px |

## 比例变化

```
修改前: 宽 960px, 输入栏高 64px   → 15:1 (扁长)
修改后: 宽 720px, 输入栏高 100px  → 7:1  (挺拔)
```

内容区收窄 240px，输入栏抬高 36px，整体从「扁宽」变为「挺拔」。

## 影响评估
- ✅ 输入区成为视觉焦点，位于屏幕下 1/3
- ✅ 窄内容区让对话流更集中、更易阅读
- ✅ 大输入框 + 大按钮，打字体验舒适
