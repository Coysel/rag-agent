## 三者的关系

```
你的 Python 代码  ──→  连接 MySQL  ──→  读写数据
      │
      └──  运行在 Linux 服务器上
      
一个典型的 Web 项目:
  Linux 机器上跑着 Python 写的服务，服务从 MySQL 读数据，处理后返回给用户
```

## 学习路线建议

```
第一周: Python 基础 (1-4章)  +  Linux 基础 (1-3章)
第二周: Python 进阶 (5-8章)  +  MySQL 入门 (1-4章)  
第三周: MySQL 进阶 (5-8章)   +  Linux 进阶 (4-6章)
第四周: 三个技术串联, 做一个小项目
```

每章的"前置依赖"标注了必须先看懂哪些内容，遇到底部出现还没学过的概念会提前引入。
